  2. Disentangled Makeup Transfer with Generative Adversarial Network
    TensorFlow code
    基于解偶表示利用编码器将图像中涉及个人属性的特征和涉及妆容的特征进行分离，随后利用解码器对于妆容和人脸的编码重建出面部图像。模型中一共包含了五个部分的损失，除了通常的对抗损失外，生成器G还包括图像的重建损失Lrec、妆容迁移后的感知损失Lper、迁移后的妆容损失Lmak以及保证一对一人脸化妆编码的重建损失Limr

  3. PSGAN: Pose-Robust Spatial-Aware GAN for Customizable Makeup Transfer
    无代码
    PSGAN主要分为三部分：妆容提炼网络（MDNet）、注意式妆容变形（AMM）模块和卸妆-再化妆网络（DRNet）。
    PSGAN 是首个能同时实现逐部分迁移、浓浅程度可控迁移和姿势稳健型迁移的基于 GAN 的方法。新提出的 AMM 模块可以调整从参照图像提取出的妆容矩阵，使其适应源图像。这个模块让 PSGAN 有能力在姿态不同的图像之间迁移妆容风格。
    (1) A learned representation for artistic style (ArXiv 2016)(风格迁移)
    (2) Arbitrary style transfer in real-time with adaptive instance normalization (2017 ICCV)(风格迁移)
